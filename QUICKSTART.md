# Quick Start Guide

## Installation (5 minutes)

### Step 1: Get the Extension
```bash
git clone https://github.com/boun-tabi-LMG/on-device-renarration.git
```

### Step 2: Load in Chrome
1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right)
3. Click **Load unpacked**
4. Select the `on-device-renarration` folder
5. You should see the 🔄 icon in your toolbar

## Quick Start (2 minutes)

### Try Text Renarration
1. Open the included `test-page.html` file in Chrome
2. Select any paragraph of text
3. Click the 🔄 button that appears
4. View the renarrated text in the popup

### Change Tasks
1. Click the 🔄 extension icon in toolbar
2. Select a different task from the dropdown
3. Try renarrating the same text to see the difference

## Troubleshooting

**Extension not working?**
- Check it's enabled in `chrome://extensions/`
- Refresh the page after enabling
- Check browser console for errors

**🔄 button not appearing?**
- Make sure you selected enough text (>10 characters)
- Check if extension is enabled in the popup
